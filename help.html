  • Having problems installing games?

    •The installer requires for the user to have administrator rights to the PC. If you do not have admin rights on your PC user account, see below:

    1. Click on Start or the Windows logo on your taskbar
    2. Right click on My Computer or Computer, select Manage
    3. On the computer management screen, click on either the + or the arrow on to expand the folder
    4. Click on the Users folder, your user name should appear on the column to the right
    5. Right click on your user name and select properties
    6. Click on the Member Of tab
    7. Click on the add button
    8. Click on advanced button
    9. Click on Find Now button
    10. Double click on administrators on the search results list
    11. Click ok until you exit out of all open windows
    12. After following the steps below, try to reinstall the games

    •If the actual installer screen still fails to start up the individual installers, you can try:

    1. If you have Windows XP click on Start on the taskbar; if on Vista or Win 7, click on the Microsoft Logo on the taskbar
    2. Choose My Computer on the menu that pops up
    3. Right click on the CD/DVD drive where the disc is inserted, usually D: and choose explore
    4. Locate the folder fscommand and double click to open
    5. Install each individual games using the installer files, usually found as "Game Title.exe" double clicking
    6. If this still does not help, copy the individual game installers to the desktop and install from there.

  • Having problems with Windows 10?

    •Windows 10 is brand new and known to have compatibility issues. However, it has built in compatibility settings. Please see below for instructions on how to activate Compatibility Mode:

    1. Right-click on the game's icon
    2. Select "Properties"
    3. Select the "Compatibility" tab
    4. Check the box next to "Run this program in compatibility mode for:"
    5. In the drop down menu, select "Windows XP (Service Pack 3)" or "Windows 7"
    6. You may also want to check mark the following other options:
    a. Disable visual themes
    b. Disable desktop composition
    c. Disable display scaling on high DPI settings
    d. Run this program as an administrator

    •Please also ensure that you have the latest driver updates. Please run Windows Update and download the latest version of Microsoft DirectX here: http://www.microsoft.com/en-us/download/details.aspx?id=3

    •Finally, depending on your computer's setup, you may also need to update your hardware-specific video drivers. To do so, please follow the below instructions:

    1. Right-click on an empty part of your desktop
    2. Select the "Screen Resolution" option
    3. Click on "Advanced settings"
    4. A new window should pop up with the name of your monitor and video card
    5. In the Adapter tab, next to your video card name, click on "Properties"
    6. Another new window should pop up. Click on the "Driver" tab.
    7. Click on "Update Driver" then "Search automatically for updated driver software"
    8. Follow the onscreen instructions

  • Trouble installing Nancy Drew games??

    •The Nancy Drew games automatically install the latest Microsoft drivers to ensure the games run properly, but this might take some time. It is recommended that you close all other applications and let the installer run until completion, which may take up to 30-40 minutes depending on your computer. The progress bar may appear stalled during the installation of these drivers. If the progress bar appears to be stalled for more than 1 hour, please follow this guide below:

    1. Click on Start or the Windows logo on your taskbar
    2. Click on My Computer or Computer
    3. Locate your DVD drive, typically labeled "(D:)"
    4. Right-click on this Drive and select "Open" or "Explore"
    5. Open the "fscommand" folder
    6. Locate the folder with the name of the game you wish to install
    7. Copy this folder from the disc to your desktop
    8. Open the newly copied folder
    9. Open the "Files" folder
    10. Open the "Game" folder
    11. Double click on the game name to start the game
    12. Repeat with the other game title
